# Glowing Skincare & Cosmetics E-commerce Website 
A fully responsive, modern e-commerce website designed for beauty and skincare products, allowing users to browse, view detailed product information, and manage their shopping cart. 

# Description
This project is a functional front-end e-commerce interface designed to provide a seamless shopping experience for beauty products. It features a modern, clean UI, focusing on high-quality visuals for skincare items. 

# Features
Fully Responsive Design: Optimized for mobile, tablet, and desktop devices.
Product Catalog: Browse various beauty and glowing skin care products.
Product Details: Detailed views with images, descriptions, and pricing.
Interactive Shopping Cart: Users can add, remove, and manage items in their cart.
User-Friendly UI: Intuitive navigation and appealing hover states. 

# Technologies Used
HTML5: Semantic markup for structure.
CSS3: Custom styling (or Tailwind/Bootstrap) for modern design.
JavaScript: Interactivity for cart management and product display 

# How to Run

Download or clone the repository.

Open the HTML file in any web browser.
